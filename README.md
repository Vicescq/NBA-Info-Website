## Python Virtual Environment Setup

### Windows:
1. cd to backend

1. python -m venv venv

2. venv/Scripts/activate

3. pip install -r requirements.txt

<br/>

### Mac:
1. cd to backend

1. python3 -m venv venv

2. source venv/bin/activate  

3. pip install -r requirements.txt

<br/>

## NodeJS Module setup
1. cd to frontend

2. npm i

<br/>
